# Rick and Morty API Project

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como atividade acadêmica utilizando:

* HTML
* CSS
* JavaScript

O objetivo do projeto é consumir dados em tempo real da API pública Rick and Morty e exibir os personagens dinamicamente na tela através de cards criados via JavaScript.

---

## 🚀 Funcionalidades

* Consumo de API utilizando `fetch`
* Criação dinâmica de cards
* Exibição de:

  * Nome
  * Imagem
  * Status
* Campo de pesquisa de personagens
* Modal com detalhes do personagem
* Tradução parcial de informações
* Layout responsivo
* Loading animado
